# Web Static

This directory contains static HTML and CSS files for the web application. These files are responsible for the visual presentation and layout of the web pages.

## Contents

- `index.html`: The main HTML file serving as the entry point of the web application.
- `styles.css`: The CSS file containing styles for formatting and styling HTML elements.

## Usage

These static files can be linked or imported into other HTML files to apply consistent styling and layout across multiple pages of the web application.
